relion_tomo_refine_defocus

This program estimates the astigmatic defoci of the individual tilt images. In tomography, the relative depth distances between the particles are known from the 3D positions of the particles. Therefore, only one defocus value is estimated for all the particles in each image. Because of the typically large number of particles in each tomogram, this value can be estimated to greater precision than in single-particle analysis.

The estimated defoci are written into the output tomogram set which replaces the given tomogram set in the output optimisation set.

Relevant program arguments:

  • --i and/or --p, --t, --mot, --ref<1/2>, --mask and --fsc: input optimisation-set and/or its components (see optimisation set).
  • --b: box size to be used for the estimation. Note that this can be larger than the box size of the reference map. A sufficiently large box size allows more of the high-frequency signal to be captured that has been delocalised by the CTF.
  • --no_astigmatism: do not refine the astigmatism, but only a scalar defocus value. This is useful if the number of particles is too small to allow for accurate astigmatism estimation. If the number is even smaller, do not use this program.
  • --j: number of threads to be used. This should be set to the number of CPU cores available.
  • --o: name of the output directory (that will be created).
